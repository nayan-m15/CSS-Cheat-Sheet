Box Model

Control sizing, spacing, and borders of elements.

width / height
Sets the width or height of an element's content area.
width: 300px;
height: auto;
margin
Sets outer spacing around an element. Accepts 1-4 values for each side.
margin: 20px 10px;
/* top/bottom: 20px, left/right: 10px */
padding
Sets inner spacing between content and border.
padding: 1rem 2rem;
border
Shorthand for border-width, border-style, and border-color.
border: 2px solid #667eea;
border-radius: 8px;
box-sizing
Determines how width and height are calculated.
box-sizing: border-box;
/* includes padding and border in width */

Display & Positioning

Control how elements are rendered and positioned.

display
Specifies the display behavior of an element.
display: block;
display: inline-block;
display: flex;
display: grid;
display: none;
position
Sets how an element is positioned in the document.
position: relative;
position: absolute;
position: fixed;
position: sticky;
top / right / bottom / left
Specifies the position offset for positioned elements.
top: 20px;
left: 0;
right: 0;
z-index
Controls stacking order of positioned elements.
z-index: 100;

Flexbox

Create flexible, responsive layouts with ease.

Container Properties

display: flex
Defines a flex container.
display: flex;
flex-direction
Defines the direction of flex items.
flex-direction: row;
flex-direction: column;
justify-content
Aligns items along the main axis.
justify-content: center;
justify-content: space-between;
justify-content: space-around;
align-items
Aligns items along the cross axis.
align-items: center;
align-items: flex-start;
align-items: stretch;
gap
Sets spacing between flex items.
gap: 1rem;
gap: 1rem 2rem; /* row-gap column-gap */

Item Properties

flex
Shorthand for flex-grow, flex-shrink, and flex-basis.
flex: 1;
flex: 0 1 auto;

CSS Grid

Create complex, two-dimensional layouts.

display: grid
Defines a grid container.
display: grid;
grid-template-columns / grid-template-rows
Defines the columns and rows of the grid.
grid-template-columns: 1fr 2fr 1fr;
grid-template-columns: repeat(3, 1fr);
grid-template-rows: auto 200px auto;
gap
Sets spacing between grid items.
gap: 2rem;
grid-gap: 1rem 2rem;
grid-column / grid-row
Specifies a grid item's size and location.
grid-column: 1 / 3;
grid-row: 1 / span 2;

Typography

Style text and control font properties.

font-family
Specifies the font for text.
font-family: 'Helvetica', Arial, sans-serif;
font-size
Sets the size of the font.
font-size: 16px;
font-size: 1.5rem;
font-size: clamp(1rem, 2vw, 2rem);
font-weight
Sets the weight (boldness) of the font.
font-weight: 400;
font-weight: bold;
line-height
Sets the height of a line box.
line-height: 1.6;
line-height: 24px;
text-align
Sets horizontal alignment of text.
text-align: center;
text-align: justify;
letter-spacing / word-spacing
Controls spacing between letters or words.
letter-spacing: 0.05em;
word-spacing: 0.2em;

Colors & Backgrounds

Apply colors, gradients, and background images.

color
Sets the text color.
color: #667eea;
color: rgb(102, 126, 234);
color: hsl(230, 75%, 66%);
background
Shorthand for all background properties.
background: #f5f5f5;
background: url('image.jpg') center/cover no-repeat;
background-gradient
Creates gradient backgrounds.
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
background: radial-gradient(circle, #fff, #000);
opacity
Sets the transparency level.
opacity: 0.8;

Transitions & Animations

Add smooth transitions and keyframe animations.

transition
Shorthand for transition properties.
transition: all 0.3s ease;
transition: color 0.2s, transform 0.3s;
transform
Applies 2D or 3D transformations.
transform: rotate(45deg);
transform: scale(1.2);
transform: translate(10px, 20px);
animation
Applies keyframe animations.
animation: slide 2s ease-in-out infinite;

@keyframes slide {
  from { transform: translateX(0); }
  to { transform: translateX(100px); }
}

Selectors & Pseudo-classes

Target specific elements with precision.

Selector Example Description
Element p Selects all paragraph elements
Class .container Selects elements with class="container"
ID #header Selects element with id="header"
Descendant article p Selects all p inside article
Child ul > li Selects direct li children of ul
:hover a:hover Selects links on mouse hover
:first-child li:first-child Selects first li element
:nth-child(n) tr:nth-child(even) Selects even numbered rows

Responsive Design

Create layouts that adapt to different screen sizes.

@media
Defines media queries for responsive design.
@media (max-width: 768px) {
  body {
    font-size: 14px;
  }
}

@media (min-width: 1024px) {
  .container {
    max-width: 1200px;
  }
}
clamp()
Sets a value that scales between a minimum and maximum.
font-size: clamp(1rem, 2.5vw, 2rem);
Viewport units
Sizes relative to the viewport.
width: 100vw;  /* viewport width */
height: 100vh; /* viewport height */
font-size: 5vmin; /* viewport minimum */

Example 2: Basic styles.css

Here's the complete CSS used to style a blog plage to learn from. A real-world example demonstrating typography, colors, transitions, and responsive design all working together.


//* is for the whole webpage
* {
  border: 1px solid red;
}

//For the all paragraphs
p {
  color: green;
}

// for all h1 headers
h1 {
  color: maroon;
}

//the . is to indicate the style of a specific class
.title {
  color: teal;
}

.uppercase {
  text-transform: uppercase;
}

//# is to indicate the style of an id 
#article-title {
  font-family: cursive;
}

//a[href*=''] is for a specific href value 
a[href*='florence'] {
  color: lightgreen;
}

a[href*='beijing'] {
  color: lightblue;
}

a[href*='seoul'] {
  color: lightpink;
}

//hover is to change the color or behavior of something on hover.
a:hover {
  color:darkorange;
}

//changing the background of an element with class heading-background
.heading-background {
  background-color: aqua;
}

//changing the color of an element with id publish-time
#publish-time {
  color: lightgray;
}

h5 {
  color: yellow;
}

.author-class {
  color: pink;
}

#author-id {
  color: cornflowerblue;
}

//if an element has an h value, an id and a class. The id will override the class and the class will override the h value. 

//this is to modify a specific header with the class destination. **Note - doesn't change all h2 values. 
h2.destination {
  font-family: Tahoma;
}

//Changes the value of the h5 that is a descendant of the description class.
.description h5 {
  color: blueviolet;
}

//changing the value of nested elements using a space. The h4 is nested within the li. 
li h4 {
  color: gold;
}

h4 {
  color: dodgerblue;
}

//Unrelated selectors can be seperated by commas and receive the same style. 
li,
h5 {
  font-family: monospace;
}
